# CISC101 Paper Summarizer – Group 66

This repository contains our final project for CISC101: **Research Paper Summarizer**.

## Structure
- **system_prompt.md** – The meta-prompt that generates a complete Summarizer System.
- **/modules/** – Contains all module files generated from the meta-prompt.
- **README.md** – Overview and instructions.
